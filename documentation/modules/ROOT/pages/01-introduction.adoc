[abstract]

== 1. Introduction: What is Ansible?

Ansible is an open-source automation tool that simplifies IT orchestration, configuration management, and application deployment. It uses a simple, human-readable language (YAML) to describe automation jobs, and requires no special coding skills to get started.

.Why Ansible?

. **Agentless**: No agents to install on managed nodes
. **Idempotent**: Run the same automation multiple times safely
. **Simple**: Human-readable YAML syntax
. **Extensible**: Thousands of community modules and collections
. **Enterprise Ready**: Backed by Red Hat with enterprise support
